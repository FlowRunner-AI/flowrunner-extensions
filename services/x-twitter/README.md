# X / Twitter FlowRunner Extension

Social media automation platform for programmatic posting, content distribution, and automated social media management on X (formerly Twitter).

## Ideal Use Cases

- Automated social media content publishing and scheduling
- Real-time notification and update broadcasting
- Social media bot development and management
- Content distribution from automated workflows
- Social media campaign automation
- Brand awareness and engagement automation
- News and announcement distribution
- Community management and interaction automation

## List of Actions

- Create Post
- Create Post with Image
