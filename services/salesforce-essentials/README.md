# Salesforce Essentials FlowRunner Extension

Essential CRM integration platform for Salesforce automation focused on core contact, lead, and campaign management workflows.

## Ideal Use Cases

- Lead generation and qualification automation
- Contact management and database maintenance
- Campaign tracking and marketing automation
- Sales pipeline management and reporting
- Customer data synchronization workflows
- File attachment and document management
- Basic CRM record creation and updates
- Simple query-based data retrieval

## List of Actions

- Add Contact to Campaign
- Add Lead to Campaign
- Create Attachment
- Create Record
- Find Record
- Find Record by Query
- Update Record
