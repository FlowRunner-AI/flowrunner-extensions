# Outlook FlowRunner Extension

Microsoft Outlook integration for comprehensive email, calendar, and contact management with OAuth2 authentication for secure access to Microsoft 365 accounts.

## Ideal Use Cases

- Automated email sending and management workflows
- Calendar event scheduling and synchronization
- Contact list management and organization
- Email draft creation and batch sending
- Meeting invitation automation
- Email folder organization and cleanup
- Contact synchronization across systems
- Calendar availability checking
- Email notification systems
- Customer communication automation
