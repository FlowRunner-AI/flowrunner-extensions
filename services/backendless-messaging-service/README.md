# Backendless Messaging Service FlowRunner Extension

Sends email and push notifications through Backendless messaging infrastructure, with support for templates, multiple recipients, and file attachments.

## Ideal Use Cases

- Send transactional emails
- Deliver push notifications to mobile apps
- Send bulk communications using templates
- Include file attachments in emails
- Notify users of important events
- Automate customer communications
- Send alerts and reminders

## List of Actions

- Send Email
- Send Push Notification

## List of Triggers

- On Push Notification Published
- On Push Notification Sent From Template
