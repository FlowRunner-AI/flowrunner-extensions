# Backendless Files Service FlowRunner Extension

Manages file storage operations and monitors file system events in Backendless, enabling AI agents to handle file uploads, organize content, and respond to file changes.

## Ideal Use Cases

- Process uploaded files automatically
- Organize files into dynamic folder structures
- Track file operations for auditing
- Trigger workflows on file uploads
- Clean up temporary files
- Manage storage quotas
- Build progressive logs and reports
- Archive and backup files
- Browse and search files in directories

## List of Actions

- Add To File
- Create Directory
- Create File
- Delete File
- List Directory

## List of Triggers

- File Copied
- File Deleted
- File Moved
- File Renamed
- File Uploaded
