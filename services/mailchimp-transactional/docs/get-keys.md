**Get Mandrill API Key** (ensure you have a Standard or Premium Mailchimp plan, as the Mandrill add-on is only available on these paid plans): <br/>

1. Log in/Sign up to your [Mailchimp](https://login.mailchimp.com/) account and go to _Profile_:<br/> ![profile](assets/profile.png)
2. Go to _Extras_ → _API Keys_:<br/> ![extras](assets/extras.png)
3. Create a Mandrill API Key:<br/> ![api-keys](assets/api-keys.png)
4. You'll be redirected to [Mandrill Login](https://mandrillapp.com/login/?referrer=%2Fsettings%2Findex%2F) page:<br/> ![login-mandrill](assets/login-mandrill.png)
5. Log in using Mailchimp, and you'll see the [Mandrill Settings](https://mandrillapp.com//settings/index/) page:<br/> ![mandrill-settings](assets/mandrill-settings.png)
6. Create a new API Key and save it because you won’t be able to see or copy this API key again.
