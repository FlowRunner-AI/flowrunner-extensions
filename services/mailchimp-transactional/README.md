# Mailchimp Transactional Email FlowRunner Extension

High-performance transactional email delivery platform with advanced template management, tracking capabilities, and comprehensive analytics.

## Ideal Use Cases

- Transactional email delivery for e-commerce and SaaS applications
- Password reset and account verification email automation
- Order confirmation and shipping notification systems
- Automated invoice and receipt email delivery
- System alerts and monitoring notification distribution
- Template-based email campaigns with dynamic content
- Email deliverability management and blacklist monitoring
- Advanced email tracking and analytics reporting

## List of Actions

- Add Rejection
- Add Template
- Get Message Info
- Get Rejections List
- Get Tags List
- Get Templates List
- Get User Info
- Ping
- Send Message
- Send With Template
