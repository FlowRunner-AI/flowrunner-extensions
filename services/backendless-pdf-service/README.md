# Backendless PDF Service FlowRunner Extension

Generates PDF documents from HTML templates using the Backendless template system, enabling AI agents to create dynamic reports, invoices, and documents.

## Ideal Use Cases

- Generate invoices and receipts
- Create reports and documentation
- Produce certificates and credentials
- Build customized documents with dynamic data
- Export data in PDF format
- Automate document generation workflows

- Generate PDF
