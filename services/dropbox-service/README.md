# Dropbox FlowRunner Extension

Complete Dropbox integration for AI-agent flows: OAuth2 authentication, file management (list, upload, download, move, copy, delete, search), sharing (links and folder members), account information, and polling triggers that react to changes in watched folders.

## Ideal Use Cases

- Ingesting client uploads or third-party deliveries from Dropbox into automated pipelines
- Archiving externally hosted assets and AI-generated content into organized Dropbox folders
- Building document processing workflows that read from and write back to Dropbox
- Managing shared folders, invitations, and access levels for collaborative teams
- Generating temporary download links to hand files to external systems without Dropbox credentials
- Monitoring folders for new uploads, new subfolders, or revised documents to trigger downstream actions
- Reorganizing, renaming, and archiving processed files based on workflow outcomes
- Reporting on Dropbox storage usage and account identity inside automation runs

## Agent Ideas

- When a **Dropbox** "On New File" trigger fires for a watched intake folder, call **Dropbox** "Get Temporary Link" and pass the URL to **PDF.co** "Parse Invoice with AI" to extract structured invoice data automatically as documents arrive.
- When a **Dropbox** "On File Modified" trigger fires on a contract folder, use **Slack** "Send Message To Channel" to notify the legal channel with the file name and a link generated by **Dropbox** "Create Shared Link" for quick review.
- After a **Dropbox** "Upload File from URL" action archives an AI-generated report, call **Notion** "Create Database Item" to log the file path, shared link, and metadata into a knowledge-base database for team discoverability.
