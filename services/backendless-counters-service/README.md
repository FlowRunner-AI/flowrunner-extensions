# Backendless Counters Service FlowRunner Extension

Monitors atomic counter operations in Backendless, enabling AI agents to track counter state changes and trigger workflows based on counter modifications.

## Ideal Use Cases

- Track usage metrics and quotas
- Monitor rate limiting counters
- Log counter modifications for auditing
- Trigger alerts when counters reach thresholds
- Coordinate workflows based on counter values
- Track inventory or resource counts

## List of Triggers

- Counter: Add And Get
- Counter: Compare And Set
- Counter: Decrement And Get
- Counter: Get And Add
- Counter: Get And Decrement
- Counter: Get And Increment
- Counter: Increment And Get
- Counter: Reset
