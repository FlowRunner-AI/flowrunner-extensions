# Backendless Data Service FlowRunner Extension

Manages database operations and monitors data changes in Backendless applications, enabling AI agents to perform CRUD operations and respond to real-time data events.

## Ideal Use Cases

- Automate data processing pipelines
- Sync data between systems
- Send notifications on data changes
- Maintain audit trails and logs
- Trigger business logic on record events
- Build event-driven architectures
- Automate data cleanup and maintenance

## List of Actions

- Delete Record In Database
- Delete Records In Database
- Find Record(s) in Database
- Save Record In Database
- Update Records With Query

## List of Triggers

- On Record Created
- On Record Deleted
- On Record Updated
