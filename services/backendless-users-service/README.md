# Backendless Users Service FlowRunner Extension

Monitors user registration events in Backendless applications, enabling AI agents to automate onboarding workflows and respond to new user signups.

## Ideal Use Cases

- Automate user onboarding processes
- Send welcome emails to new users
- Initialize user profiles and preferences
- Trigger account setup workflows
- Log user registrations for analytics
- Integrate with CRM systems
- Create audit trails for user activity

## List of Triggers

- On New User Registered
