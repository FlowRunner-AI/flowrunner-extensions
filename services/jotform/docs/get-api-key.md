1. Go to https://www.jotform.com/
2. Sign Up
3. Go to [API Settings](https://jotform.com/myaccount/api)
4. Create API Key and update permission to **Full Access**
5. Go to [Data Settings](https://eu.jotform.com/myaccount/data)
6. Toggle the option _"Store my data only in Europe (Germany)"_ if you prefer your data to be stored in the European region.
   If this is not enabled, your data will be stored in the USA by default
7. If you've set your API key but receive an _'Unauthorized'_ error when making a request, try logging in to https://jotform.com and then retry the request

Their docs: https://api.jotform.com/docs/#overview
