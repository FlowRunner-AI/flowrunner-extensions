# Jotform FlowRunner Extension

A comprehensive form management integration that enables automated form creation, question management, and submission data collection. This extension provides seamless integration with Jotform's API for building dynamic surveys, collecting responses, and managing form workflows through automated processes.

## Ideal Use Cases

- Building AI agents that create dynamic forms based on user requirements
- Automating survey creation and distribution for customer feedback systems
- Creating event registration workflows with automated form generation
- Implementing data collection systems with automated response processing
- Building analytics dashboards that pull form submission data automatically
- Managing multi-step form workflows with conditional question logic
- Developing customer onboarding processes with automated form sequences

## List of Actions

- Add Questions to Form
- API Request
- Create New Form
- Get Form Questions
- Get Form Submissions
- Get Monthly User Usage
- Get User Submissions
