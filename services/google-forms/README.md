# Google Forms FlowRunner Extension

Complete Google Forms management for creating, updating, and processing form responses. Build dynamic survey workflows, manage form structures, and extract valuable data from submissions.

## Ideal Use Cases

- Creating dynamic surveys and forms for customer feedback collection
- Managing form structures with automated field updates and modifications
- Processing form responses for lead generation and customer insights
- Building automated survey workflows with conditional logic and branching
- Integrating form data with CRM systems and marketing automation
- Creating feedback collection systems for product and service improvement
- Managing large-scale data collection campaigns with form validation
- Automating form response analysis and reporting workflows

## List of Actions

- Change Field Position
- Create Form
- Delete Field Form
- Delete Form
- Get Form Details
- Get Form Response by ID
- Get Form Responses List
- Get Forms List
- Update Form Advanced
