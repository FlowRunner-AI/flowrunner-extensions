# Asana FlowRunner Extension

A comprehensive project management integration that enables automated task creation, project management, and team collaboration workflows. This extension provides full control over Asana workspaces, projects, tasks, and team communication through automated processes.

## Ideal Use Cases

- Automating task creation and assignment based on external triggers
- Building AI agents for project management and workflow optimization
- Creating automated project templates and task standardization systems
- Managing team collaboration with automated comment and file attachment workflows
- Implementing custom task tracking and progress monitoring systems
- Building integrated project management dashboards and reporting tools
- Automating project planning and resource allocation processes

- Add Comment
- Add Task to Section
- Attach File

- Create Section
- Create Subtask

- Find Project by Name
- Find Section by Name
- Find Task in Project

- Get Section by ID
