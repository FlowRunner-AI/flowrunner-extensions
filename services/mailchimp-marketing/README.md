# Mailchimp Marketing FlowRunner Extension

Comprehensive email marketing automation platform for audience management, campaign creation, and subscriber engagement tracking.

## Ideal Use Cases

- Email marketing campaign management and automation
- Subscriber list building and audience segmentation
- Newsletter distribution and subscriber engagement tracking  
- E-commerce customer lifecycle marketing
- Event-based email automation and trigger campaigns
- A/B testing for email campaign optimization
- Customer behavior tracking and analytics
- Multi-channel marketing automation workflows

## List of Actions

- Add Campaign
- Add Event
- Add Member Note
- Add Member Tag
- Add Member To List
- Add Or Update List Member
- API Request
- Archive List Member
- Create Tag
- Get Campaign Info
- Get Campaign Report
- Get Clicks Report
- Get Lists Info
- Remove Member Tag
- Search Campaign
- Search Customer
- Search Member
- Search Tag
- Send Campaign
- Unsubscribe Or Delete List Member
