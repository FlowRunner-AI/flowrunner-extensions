# Xero FlowRunner Extension

A comprehensive integration with Xero accounting software that enables automated financial workflows. This extension provides full access to contacts, invoices, payments, bank transactions, credit notes, quotes, purchase orders, items, accounts, and financial reports, with real-time and polling triggers for event-driven automation.

## Ideal Use Cases

- Automating invoice creation and payment tracking for recurring billing workflows
- Building AI agents that sync customer data between CRM systems and Xero
- Creating automated payment reminders and collection workflows based on aging reports
- Implementing real-time notifications when invoices are paid or overdue
- Developing expense tracking automation with bank transaction categorization
- Setting up automated quote-to-invoice conversion workflows
- Building supplier payment automation with purchase order management
- Automating financial reporting and dashboard updates
- Creating customer onboarding workflows that provision contacts and payment terms
- Implementing multi-currency transaction processing for international businesses
