# Calendly FlowRunner Extension

Automate scheduling workflows and meeting management with Calendly's powerful booking system. Create one-off meetings, manage cancellations, and trigger workflows based on booking events.

## Ideal Use Cases

- Automating follow-up workflows when prospects book meetings
- Creating personalized booking pages for VIP clients and targeted outreach
- Managing cancellations and no-shows with automated retention campaigns
- Triggering CRM updates when meetings are scheduled or cancelled
- Generating exclusive single-use booking links for special events
- Automating confirmation emails and meeting preparation workflows
- Integrating routing form submissions with lead qualification processes
- Streamlining demo scheduling with automated calendar management

## List of Actions

- Cancel Scheduled Event
- Create One-Off Meeting
- Create Single-Use Scheduling Link

## List of Triggers

- On Invitee Canceled
- On Invitee Created
- On Mark Invitee as No-Show
- On New Routing Form Submitted
- On Unmark Invitee as No-Show
